GET /api/v1/shows

Retrieves a high-level list of all available shows in the library.

[
  {
    "id": "816d2993-...",
    "title": "Batman: Caped Crusader",
    "type": "series",
    "poster": "https://...",
    "addedAt": 1770381997122,
    "seasonCount": 1
  }
]

GET /api/v1/shows/:id

Retrieves the complete data structure of a show, including all seasons, episodes, and resolved download links.

{
  "id": "...",
  "title": "...",
  "seasons": [
    {
      "number": "1",
      "episodes": [
        {
          "title": "Episode 1",
          "links": [ { "quality": "1080p", "finalLink": "https://mega.nz/..." } ]
        }
      ]
    }
  ]
}

GET /api/v1/search?q={query}

Filter the library by title. Partial matches are supported.

POST /api/v1/ingest

Request the system to scrape and ingest a new show. Returns a task ID for tracking.

Body: { "url": "https://animesalt.top/..." }

{
  "message": "Ingestion task created",
  "taskId": "fa7afdf1-..."
}
